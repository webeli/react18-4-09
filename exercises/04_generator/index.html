Now the time has come to create your own Redux application! Like the Redux example we will focus on just using Redux, no React. As you build your app, refer to the slides and take the same steps as in the chapter!

We are going to create a character generator for a role playing game. To begin, make a slider where the user can select a strength value between 1 and 5! As in the example, our app state will just be a single number.

Some regular HTML and DOM good-to-knows:

  • The HTML for the input would be something like <input type="range" value="3" min="1" max="5">.
  • In an event listener, get to an input value by doing e.target.value (if you named your parameter e, that is)
