Now we are going to add a stamina slider. As our state already is an object that merely means adding another property, but the code repetition is quickly getting out of hand.

Therefore the time has come to curb that! Instead of adding a third action creator, make a single one that takes the name of the characteristic (strength/intelligence/stamina) and the value. This also means that we now should only have a single action type.

If you have strong jQuery fu, you can earn bonus points by using event delegation to only set a single event handler for all sliders! Additionally, for gymnastics points, make your UI updater function loop over id:s to set the spans instead of touching them individually.

Two more ES6 tricks that might come in handy now:

  • if key and value are named the same, we can omit the value: let name = "me"; let obj = {name};
  • set a dynamic key like this: let propname = 'foo'; let obj = {[propname]: 42};
