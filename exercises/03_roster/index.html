Let's put our new-found React skills to the test by building a Roster app! In it we will enter the names of our employees together with an assessment of their skill. As a humble first step we create a UserForm component where you can enter a name. So open up yourcode1.js and get to work!

See how it is supposed to work in the column to the right, and also check how the Tester component is set up in your code file.

Create UserForm using the uncontrolled pattern. That means...

  • Your form input controls should have defaultValue and refs
  • In the button callback you collect the values using the refs!
